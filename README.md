### Repository Contents Summary
Based for now on the `FF_Visual.py` file provided, the repository is structured to include:

* **`FF_Visual.py`**: Contains the `InteractiveFirefighter` class with features for toggling modes (Set Root vs. Protect), undoing actions, and resetting the game.
* **`Fire-Fighter-Enviroment.ipynb`**: A Jupyter notebook demonstrating how to initialize a graph (e.g., uses a hexagonal lattice), and build the interactive canvas.

### Installation & Quick Start
To use the tool, users will need to install `networkx`, `matplotlib`, `numpy`, and `ipympl`. The visualizer is best used in a Jupyter environment with the `%matplotlib widget` backend enabled to support interactive button clicking and node selection.
